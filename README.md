 Hospital Patient Analytics Dashboard

## Project Overview
This project analyzes hospital patient data using Microsoft Excel. 
The goal of this project is to understand patient details, department performance, and billing information through data analysis and dashboards.

## Dataset
The dataset contains the following fields:
- Patient ID
- Patient Name
- Age
- Gender
- City
- Department
- Doctor Name
- Bill Amount
- Payment Status
- Appointment Time

## Data Cleaning
The following data cleaning steps were performed:
- Handled missing values
- Standardized appointment time format
- Organized patient information
- Created structured tables for analysis

## Analysis Performed
- Department wise patient analysis
- Billing amount analysis
- Payment status breakdown
- City based patient distribution

## Dashboard
An interactive Excel dashboard was created using:
- Pivot Tables
- Charts
- Slicers for filtering data

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning Techniques
- Data Visualization

## Project Outcome
This dashboard helps in understanding hospital performance, patient distribution, and billing insights in a clear visual format.
